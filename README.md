#  Module-1 Day-1 SEB
## AIM: 
To write a C program to read only positive integer value from the user and display the same value as output.

## Program:
```c
#include <stdio.h>
int main()
{
   unsigned int num;
    scanf("%d",&num);
    printf("%u",num);
    
  
  return 0;
}
```
